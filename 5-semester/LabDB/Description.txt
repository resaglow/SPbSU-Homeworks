Описание предметной области "Лаборатория, её исследования" 

База данных построена вокруг деятельности лаборатории, разработок/исследований (есть разные области исследований) для организации деятельности данной лаборатории.
Соответственно есть исследователи, которые как-то между собой договариваются, что именно каждый из них будет делать в рамках данного исследования,
и в итоге работают каждый более менее независимо от остальных (над своей частью исследования).
Работают в кабинетах, в каждом кабинете есть оборудование, специфичное для данного кабинета, в кабинете могут работать несколько учёных / учёный может 
работать в разные дни в разных кабинетах. Ученый записывает свое "расписание" в систему, в ней с этого момента хранится информация о том, по каким дням и в каких
кабинетах работает данный ученый.
Каждый исследователь владеет каким-то количеством материалов (сырья), которое может использоваться при наличии для любых исследований.
У лаборатории есть спонсоры, у спонсоров есть "единицы финансирования" (sponsorship), относящиеся к некоторым исследованиям, 
у исследования может быть несколько sponsorship-ов.
Предполагается по имеющимся данным выбирать спонсоров, которые наиболее активно финансировали разработки лаборатории, 
и, например, каким-то образом поощрять данных спонсоров.
